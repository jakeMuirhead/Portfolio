Davie’s Burgers – Custom Burger Builder

  • Mobile-Friendly Design – Optimized for various screen sizes, ensuring seamless interaction on all devices.

This project highlights expertise in form handling, user interface design, and accessibility, making it a strong foundation for interactive web applications.
